以下是要作为一名优秀的前端工程师所要具备基本的能力：

HTML

超文本标记语言,标准通用标记语言下的一个应用。"超文本"就是指页面内可以包含图片、链接,甚至音乐、程序等非文字元素

CSS 3

最新的css标准。高级的浏览器可以识别的一些高级行为样式,如：圆角:border-radius;阴影:box-shadow等诸多特性。

HTML 5

下一代HTML标准。增加了canvas、audio等新标签。

JAVASCRIPT

是一种直译式脚本语言,是一种动态类型、弱类型、基于原型的于洋,内置支持类型。

PS

即Photoshop,主要处理以像素所构成的数字图像。使用其众多的编修与绘图工具。

JQUERY

是一个快速的,简洁的JavaScript库,由美国人John Resig创建。
